Detailed Description

Renderers can be associated with zero or more resources.

Resources are extra components which might be required rendering, such as textures or uniform buffers.

Resources can vary wildly in their nature depending on what sort of rendering you'd like to do. If you don't provide any resources to a renderer, it will simply draw its input to its output.

However, if you have extra data which you would like to use in shaders, then you can create resources and pass them to the IRendererBuilder so it can be used by the renderer. The most common use-case is for adding model, view and projection data via uniform buffers.
