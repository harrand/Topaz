◆ get_default_monitor()

MonitorInfo tz::get_default_monitor ( )

Retrieve the default/primary monitor.

This is usually the one with global UI elements such as the task or menu bar.

Precondition
At least one monitor must be connected, otherwise the behaviour is undefined. To retrieve monitor information safely if you cannot guarantee this, retrieve the first result of get_monitors() if its not empty.

◆ get_monitors()

tz::BasicList< MonitorInfo > tz::get_monitors ( )

Retrieve a list of all monitors, sorted by their ids.

If no monitors are detected, returns an empty list.

Note
The default monitor (see get_default_monitor()) is guaranteed to be the first element in this list.
