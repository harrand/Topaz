◆ DepthComparator

enum class tz::gl::vk2::DepthComparator
strong

When two fragments depth values are compared aka comp(lhs, rhs), what should the return expression be?

Enumerator
AlwaysFalse 
  • Return false.
LessThan 
  • Return lhs < rhs.
EqualTo 
  • Return lhs == rhs.
LessThanOrEqual 
  • Return lhs <= rhs.
GreaterThan 
  • Return lhs > rhs.
NotEqualTo 
  • Return lhs != rhs.
GreaterThanOrEqual 
  • Return lhs >= rhs.
AlwaysTrue 
  • Return true.
