constexpr KeyInfo tz::peripherals::keyboard::get_key ( char  representation)
constexpr

Attempt to retrieve the key information corresponding to a typed character.

If the char does not correspond to a key, the null-key is instead returned. See key_null.

For example, ‘get_key('A’).code == KeyCode::A`
