Detailed Description

template<tz::message M, bool thread_safe = true>
class tz::message_receiver< M, thread_safe >

Conditionally thread-safe message receiver.

Template Parameters
MMessage Type. Must satisfy tz::message<M>
thread_safeWhether the object should lock an internal mutex. Defaults to true, but you might want to set this to false for a thread-local receiver, for example.

Invoked when a message is processed.

You must override this. Not thread safe.

Note
If processing a message causes the receiver to send a new message to itself, this is safe, but will not be processed until the next invocation to process_messages.
