Detailed Description

A collection of platform-agnostic core interfaces.

This is the only module containing members of the Initial Group. The Initial Group is a group of functions and structures which can be invoked before tz::initialise().

◆ initialise()

void tz::initialise ( GameInfo  game_info,
ApplicationType  app_type = ApplicationType::WindowApplication,
WindowInitArgs  wargs = tz::default_args 
)

Initialise Topaz.

Precondition
If any other Topaz code not within the "Initial Group" has been invoked before this, then the behaviour of the program is undefined. To know if something is within the Initial Group, check their doucmentation notes.
Postcondition
If tz::terminate() is not called before the end of program runtime, then the behaviour of the program is undefined.

This initialises the render-api backend. Also creates a window unless ApplicationType::Headless is specified. Note that for ApplicationType::HiddenWindowApplication, a window is created but it is invisible.

Parameters
game_infoInformation about the application
app_typeDescribes the anatomy of the application.

◆ rotate()

Mat4 tz::rotate ( Vec3  rotation)

Generates a matrix which performs the following three transformations, in chronological order:

  • Rotates the result rotation[0] radians about the x-axis.
  • Rotates the result rotation[1] radians about the y-axis.
  • Rotates the result rotation[2] radians about the z-axis.

◆ scale()

Generates a matrix which performs the following transformations at once:

  • Multiplies each value by scale[0] in the x-direction.
  • Multiplies each value by scale[1] in the y-direction.
  • Multiplies each value by scale[2] in the z-direction.

◆ terminate()

void tz::terminate ( )

Terminate Topaz.

Precondition
If tz::initialise has not been invoked before this, then the behaviour of the program is undefined.
Postcondition
If anything in tz is invoked after this, then the behaviour of the program is undefined.

This destroys the render-api backend. If a window was created previously by tz::initialise, then it will be destroyed here.
